# 🚀 FlyRank Machine Learning Internship

> A complete documentation of my journey through the **FlyRank Machine Learning Internship**, where I work with real-world search data, machine learning, prompt engineering, AI workflows, and portfolio development.

---

## 📌 About the Internship

The **FlyRank Machine Learning Internship** is a hands-on, project-based internship designed to teach practical Machine Learning using real anonymized search data.

Throughout the internship I learn how to:

- Frame business problems as Machine Learning tasks
- Work with large search datasets
- Build reproducible data science workflows
- Apply Prompt Engineering techniques
- Use AI tools effectively
- Develop a professional technical portfolio
- Think like an ML Engineer rather than simply training models
